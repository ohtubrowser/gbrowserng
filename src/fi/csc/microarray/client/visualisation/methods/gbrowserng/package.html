Package contains main class for running of program and global variables used by most packages. Package includes 2 classes written / modified by team which are fully tested and commented: GenomeBrowserNG and GlobalVariables. Main class for running program, GenomeBrowserNG, initializes all needed components and loads all needed data from files regarding genome and links between chromosomes. GlobalVariables contains variables used throughout program by many classes, and as such is placed in the topmost package. Variables contained include size of screen.
